## What this repo does
1) Publishes a technical authority layer (frameworks + automation + experiments).
2) Includes **structured data (JSON-LD)** on every page for entity consistency.
3) Runs a **monthly GitHub Actions refresh** that generates a new crawlable page in `/maintenance/YYYY-MM/`.

## Deploy (GitHub Pages)
- Push this repo to GitHub under your account.
- In GitHub: Settings → Pages → Build and deployment → Source = `Deploy from a branch`
- Branch = `main` (or `master`), folder = `/ (root)`
- Wait for Pages to publish.
